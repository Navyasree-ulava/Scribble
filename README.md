# Scribble

Scribble is a **real-time collaborative drawing application** that allows multiple users to draw and interact on a shared canvas simultaneously. Built using **WebSockets** and **Express**, the application ensures **low-latency updates** and **consistent canvas state** across all connected clients.

The project is designed with a **monorepo architecture** to cleanly separate frontend, backend, and shared configuration, making it scalable and easy to maintain.

## Features  
✅ Real-time drawing updates using WebSockets  
✅ Backend powered by Express for handling connections  
✅ Efficient state management for synchronized collaboration  
✅ Simple and intuitive UI for easy drawing  

## Tech Stack  
- **Frontend:** Next.js
- **Backend:** Node.js, Express  
- **Real-Time Communication:** WebSockets (native WebSockets) 
- **Monorepo Mangement:** TurboRepo
- **Package Manager:** pnpm
